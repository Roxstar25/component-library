# Vanilla JS Component Library

A dark-mode, glassmorphism UI component library built with **vanilla HTML, CSS, and JavaScript**. Zero dependencies. Zero frameworks.

**Live demo:** https://roxstar25.github.io/component-library/

## Components

| # | Component | Features |
|---|-----------|----------|
| 1 | **Modal Dialog** | Overlay blur, scale animation, Escape to close |
| 2 | **Tab Panel** | Sliding pill indicator, content slide transition |
| 3 | **Accordion** | Smooth height animation, rotating icon |
| 4 | **Image Carousel** | Auto-play, dot navigation, prev/next arrows |
| 5 | **Toast Notifications** | Auto-dismiss, slide-in animation, 3 variants |
| 6 | **Dropdown Menu** | Glassmorphism, scale + fade entrance |

## Design

- Dark mode with animated gradient background
- Glassmorphism cards (backdrop-filter blur)
- iOS-style blue accent (#0a84ff)
- Smooth cubic-bezier transitions

## Tech

- HTML5 semantic markup
- CSS custom properties, Grid, Flexbox
- Vanilla JavaScript (DOM manipulation, event handling)
- GitHub Pages deployment
